## üì¶ Install

```bash
$ git clone https://github.com/Sakchai/vite-react-ts-web-admin.git
$ cd vite-react-ts-web-admin


# yarn
$ yarn install
$ yarn dev

# npm
$ npm run install
$ npm run dev
```
